A simple browser-based number guessing game built with HTML, CSS, and JavaScript. The game randomly selects a number between 1 and 100, and the player must guess the correct number using feedback prompts that indicate whether their guess is too high or too low. It also tracks the number of attempts and displays the guesses made during the game.

The project uses a clean centred card layout, a gradient background, and basic form interaction to practise DOM manipulation, conditional logic, user input handling, and simple game state management.

Tech used: HTML, CSS, JavaScript
Focus: DOM manipulation, conditionals, random numbers, user input, and basic UI styling.
